ChannelDeck

A focused macOS IPTV player for Xtream-style accounts, with live categories, fast channel browsing, full-screen playback, and local credential storage.

Available now

Built for live IPTV playback

Xtream-style login

Connect with server URL, ID, password, and the provider's player API.

Live categories

Loads live categories and channel lists, with search and category filtering.

Secure local storage

Stores passwords in the macOS Keychain and keeps public builds credential-free.

Playback controls

Supports play, pause, stop, previous/next channel, copy URL, and open URL fallback.

Full-screen player

Includes a theater-style full-screen mode with Escape to exit.

Keyboard first

Shortcuts cover reload, playback, channel navigation, full screen, and panel toggles.

Roadmap

  • iOS and iPadOS apps with shared account and playback model.
  • Real recorded demo for the website and README.
  • Developer ID signing, notarization, and a cleaner macOS installer flow.
  • EPG guide support with current and upcoming program metadata.
  • Favorites, pinned channels, and persisted recently played history.
  • VOD and series support from compatible provider APIs.
  • Picture-in-picture and improved external player handoff.
  • Import and export for M3U playlists where legally supported.

Content and credentials

ChannelDeck is a client only. It does not include IPTV content, subscriptions, provider credentials, playlists, or authorization to access streams.
